# Day 7 - Teleporter

> Written in Kotlin

This is a map walking algorithm. Given a map which contains a start position `S` and a series of
splitters `^`, then we want to know how many exit points we have from the map if a light is shone
down from `S`.

> The travel path is always **down** from the starting point.

Regarding the input:

* All lines are of an equal length
* The only characters that can appear include `^.S`
* There is only ever a single `S` in the puzzle block

For example, given the following maze:

```text
.......S.......
...............
.......^.......
...............
......^.^......
...............
.....^.^.^.....
...............
....^.^...^....
...............
...^.^...^.^...
...............
..^...^.....^..
...............
.^.^.^.^.^...^.
...............
```

The light shines as follows:

```text
.......S.......
.......|.......
.......^.......
...............
......^.^......
...............
.....^.^.^.....
...............
....^.^...^....
...............
...^.^...^.^...
...............
..^...^.....^..
...............
.^.^.^.^.^...^.
...............
```

And splits down at each splitter:

```text
.......S.......
.......|.......
......|^|......
...............
......^.^......
...............
.....^.^.^.....
...............
....^.^...^....
...............
...^.^...^.^...
...............
..^...^.....^..
...............
.^.^.^.^.^...^.
...............
```

Until the final travel diagram ends up as:

```text
.......S.......
.......|.......
......|^|......
......|.|......
.....|^|^|.....
.....|.|.|.....
....|^|^|^|....
....|.|.|.|....
...|^|^|||^|...
...|.|.|||.|...
..|^|^|||^|^|..
..|.|.|||.|.|..
.|^|||^||.||^|.
.|.|||.||.||.|.
|^|^|^|^|^|||^|
|.|.|.|.|.|||.|
```

With `9` exit points and with the light being uniquely split `21` times.

## Part two

Part two also asks how many possible paths there are for our exit paths to have taken. For example,
in the example above, the below is one possible path:

```text
.......S.......
.......|.......
......|^.......
......|........
.....|^.^......
.....|.........
....|^.^.^.....
....|..........
...|^.^...^....
...|...........
..|^.^...^.^...
..|............
.|^...^.....^..
.|.............
|^.^.^.^.^...^.
|..............
```

And there are `40` in total.

## This script

This script uses Kotlin and requires compilation before it can be run:

```shell
mvn clean package
java -jar <program> input.txt
```

To enable debug logging, please provide the additional option of `-Dlog.level=debug`, i.e.:

```shell
java -Dlog.level=debug -jar <program> input.txt
```

A test package has been provided which can be run with:

```shell
mvn clean test
```
